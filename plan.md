# Badminton Class Plan

Goal: introduce fundamentals safely, build confidence, and finish with simple rallies.

1. Welcome + safety briefing (5 min)
   - Class goals, court rules, injury prevention reminders.
2. Warm-up + mobility (10 min)
   - Light cardio, shoulder/hip mobility, dynamic stretches.
3. Grip + ready stance + swing basics (10 min)
   - Forehand grip, split step, short swing path without shuttle.
4. Footwork fundamentals (15 min)
   - Base position, 2-step movement, recover to center.
5. Serve and return basics (10 min)
   - Low serve for doubles, simple return to center.
6. Controlled rally drills (15 min)
   - Partner feeds, short court rallies, focus on consistency.
7. Cool-down + recap (5 min)
   - Static stretching, key takeaways, homework tips.

# 羽毛球课程计划

目标：安全地掌握基础、建立信心，并以简单对拉收尾。

1. 欢迎与安全说明（5分钟）
   - 课程目标、场地规则、伤害预防提醒。
2. 热身与关节活动（10分钟）
   - 轻度有氧、肩髋活动、动态拉伸。
3. 握拍 + 准备姿势 + 挥拍基础（10分钟）
   - 正手握拍、分腿垫步、无球短挥拍路径。
4. 步法基础（15分钟）
   - 基准位、两步移动、回中。
5. 发球与接发基础（10分钟）
   - 双打低发球、简单接发回中。
6. 控制性对拉练习（15分钟）
   - 伙伴喂球、半场对拉、强调稳定性。
7. 放松与复盘（5分钟）
   - 静态拉伸、关键要点、课后小练习。
