# Badminton Coach Project - Agent Guidelines

## Project Overview
This project contains training materials for adult beginner badminton players (30+ years old). All documents are instructional content designed to guide systematic badminton learning.

---

## Document Formatting Requirements

### 1. Bilingual Format (Mandatory)
All markdown files in this project **MUST** be written in bilingual format:
*   **English section first** - Complete and self-contained.
*   **Chinese section second** - Complete translation, not just a summary.
*   Use a horizontal rule (`---`) to separate the two language sections.

### 2. Structure Example
```markdown
# English Title

[Full English content here]

---

# 中文标题

[完整中文内容在此]
```

### 3. Target Audience
*   Adults aged 30 and above.
*   Complete beginners with no prior badminton experience.
*   Focus on injury prevention and sustainable learning.

### 4. Content Tone
*   Clear and instructional.
*   Avoid overly technical jargon; explain terms when necessary.
*   Emphasize safety (especially for joints: knees, wrists, ankles).

---

## File Naming Conventions
*   Use lowercase with hyphens (e.g., `first-step.md`, `learning-path.md`).
*   Names should be descriptive and in English.

# 羽毛球教练项目 - Agent 规范

## 项目概述
本项目包含面向成人羽毛球初学者（30岁以上）的培训材料。所有文档均为指导系统学习羽毛球的教学内容。

---

## 文档格式要求

### 1. 双语格式（强制）
本项目中的所有 Markdown 文件**必须**采用双语格式：
*   **英文部分在前** - 内容完整且独立。
*   **中文部分在后** - 完整翻译，而非摘要。
*   使用水平分割线 (`---`) 分隔两个语言部分。

### 2. 结构示例
```markdown
# English Title

[完整英文内容]

---

# 中文标题

[完整中文内容]
```

### 3. 目标受众
*   30岁及以上的成年人。
*   完全没有羽毛球经验的初学者。
*   注重运动损伤预防和可持续学习。

### 4. 内容语调
*   清晰且具有指导性。
*   避免过于专业的术语；必要时进行解释。
*   强调安全性（尤其是关节：膝盖、手腕、脚踝）。

---

## 文件命名规范
*   使用小写字母和连字符（例如：`first-step.md`、`learning-path.md`）。
*   文件名应具有描述性，并使用英文。
